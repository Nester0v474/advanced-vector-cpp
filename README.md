# Advanced Vector

Продвинутая реализация контейнера vector с полной поддержкой STL и оптимизациями производительности.

## Описание

Полнофункциональная реализация контейнера vector, совместимая со стандартными алгоритмами STL. Контейнер поддерживает все основные операции, включая итераторы, аллокаторы, оптимизации перемещения и обеспечивает exception safety.

## Функциональность

- Все стандартные операции vector (push_back, pop_back, insert, erase и т.д.)
- Итераторы (random access iterator)
- Move semantics для оптимизации производительности
- Exception safety (strong exception guarantee)
- Совместимость со стандартными алгоритмами STL
- Оптимальное управление памятью

## Технологии

- **C++17** - основной язык программирования
- **Шаблоны** - для обобщенного программирования
- **Итераторы** - random access iterator
- **Move semantics** - для оптимизации производительности
- **Placement new** - для управления памятью
- **Exception safety** - strong exception guarantee
- **RAII** - управление ресурсами

## Использование

### Базовое использование

```cpp
#include "vector.h"

Vector<int> vec;

// Добавление элементов
vec.push_back(1);
vec.push_back(2);
vec.push_back(3);

// Доступ к элементам
std::cout << vec[0] << std::endl; // 1
std::cout << vec.size() << std::endl; // 3
```

### Использование с STL алгоритмами

```cpp
#include <algorithm>
#include "vector.h"

Vector<int> vec = {5, 2, 8, 1, 9};

// Сортировка
std::sort(vec.begin(), vec.end());

// Поиск
auto it = std::find(vec.begin(), vec.end(), 5);

// Трансформация
std::transform(vec.begin(), vec.end(), vec.begin(), 
               [](int x) { return x * 2; });
```

### Move semantics

```cpp
Vector<int> vec1 = {1, 2, 3, 4, 5};
Vector<int> vec2 = std::move(vec1); // Эффективное перемещение
```

## Особенности реализации
